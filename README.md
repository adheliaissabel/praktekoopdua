# Praktek OOP Dua 🚀

Repositori ini dibuat untuk mendokumentasikan tugas, latihan, atau proyek praktikum kedua pada mata kuliah **Pemrograman Berorientasi Objek (Object-Oriented Programming - OOP)**.

## 📌 Deskripsi Proyek
Proyek ini berfokus pada implementasi konsep-konsep lanjutan dari OOP. Di dalam repositori ini, terdapat penerapan beberapa pilar utama OOP serta fitur-fitur pendukungnya untuk menyelesaikan studi kasus yang telah ditentukan.

## 💻 Konsep OOP yang Diimplementasikan
Proyek ini menerapkan beberapa prinsip utama dalam OOP, antara lain:
*   **Encapsulation (Enkapsulasi):** Menyembunyikan data sensitif menggunakan *access modifier* (seperti `private`) dan menyediakannya melalui *getter* dan *setter*.
*   **Inheritance (Pewarisan):** Membuat struktur kelas hierarki di mana kelas anak (*subclass*) dapat mewarisi properti dan metode dari kelas induk (*superclass*).
*   **Polymorphism (Polimorfisme):** Kemampuan suatu objek untuk memiliki banyak bentuk, baik melalui *Method Overriding* maupun *Method Overloading*.
*   **Abstraction (Abstraksi):** Menyembunyikan detail implementasi menggunakan *Abstract Class* atau *Interface*.
